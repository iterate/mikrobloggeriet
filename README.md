# OLORM

## Installing the CLI

First, please install [babashka] and [bbin].

Then, install the `olorm` CLI:

    bbin install ./cli

You should then be able to invoke the CLI:

    $ olorm -h
    Available subcommands:

      olorm create
      olorm help
      olorm repo-path
      olorm set-repo-path

[babashka]: https://babashka.org/
[bbin]: https://github.com/babashka/bbin
