# Mikrobloggeriet content server

This server powers [mikrobloggeriet.no].

## Run a local development server

In a REPL, run:

``` clojure
user> (start!)
```

## Run Clerk

In a REPL, run:

``` clojure
user> (clerk-start!)
```

Want to learn how to use Clerk?
Read the [Clerk book].
